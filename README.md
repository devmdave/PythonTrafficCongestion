# Traffic Flow Congestion
This software is made in python using opencv-python, pandas and various other libraries including the key role of yolov8 ultralytics.








## Run Project Locally


to run this project you need some of the libraries installed on your system

you need an virtual enviorment too 
Let's Create using a Command.
```bash
    virtualenv venv
```

after creating venv move to venv\Scripts
To activate venv run a batch file named activate.bat using terminal.

```bash
    activate.bat 
```

after activating install required libraries using requirements.txt
 
Let's do it too with a Command.
```bash
    pip install -r requirements.txt
```

after installing required packages.
Run a file naemd main.py 
```bash
python main.py 
```
